# FBLA Coding & Programming 2025-26 Submission
Locating businesses around our community!

## Dependencies
Make sure that python and pip is installed and working.

To verify run in your terminal:
```bash
python --version
pip --version
```


Install the following python packages if not already installed:
- flask
- webview

On your terminal, run:
```bash
pip install webview
pip install flask
pip install requests
```

## Run project
You can either:
1. Run project from terminal.
   ```bash
   python3 run_app.py
   ```

2. Run run_app.py in file manager and open in python launcher.
